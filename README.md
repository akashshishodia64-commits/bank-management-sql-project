# 🏦 Bank Management System (SQL Project)

## 📌 Overview
This project is a relational database system built using MySQL to manage banking operations such as customers, accounts, and transactions with financial analytics.

---

## ⚙️ Features
- Customer Management
- Account Management
- Transaction Tracking
- Financial Analysis using SQL
- Views and Indexing for performance optimization

---

## 🛠️ Tech Stack
- MySQL
- SQL (DDL, DML, Joins, Aggregations)

---

## 📊 Key Insights Queries
- Top 5 Rich Customers
- Monthly Transaction Trends
- Deposit vs Withdraw Analysis
- Customer Activity Level
- Account-wise Transaction Summary

---

## 📂 Files Included
- bank_management.sql (Full database script)

---

## 🚀 How to Use
1. Open MySQL Workbench
2. Run the SQL file
3. Execute queries step by step
